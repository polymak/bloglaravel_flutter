# BlogLaravelFlutter API

Backend API Laravel 10 pour une application mobile Flutter et un site web de blog/actualités. Cette API REST fournit une solution complète de gestion de contenu avec authentification sécurisée, CRUD d'articles et gestion d'utilisateurs.

## Technologies
- Laravel 10
- Flutter
- MySQL
- API REST
- Laravel Sanctum
- PHP 8.1+
